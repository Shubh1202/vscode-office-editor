## 2. 🚀 Growth Strategy (User Acquisition)

### Target Audience
- Data Scientists (CSV/XLSX inspection)
- Technical Writers (DOCX editing)
- Developers who hate context switching

### Platforms & Posting
1. **Reddit (r/vscode, r/programming, r/webdev)**
   - *Angle*: "I got tired of opening Excel just to check a cell value, so I built this."
2. **Dev.to / Hashnode**

## 3. 💰 Monetization Roadmap

### Phase 1: Growth (100% Free)
Keep all current features free to build a user base (goal: 5,000 installs).

### Phase 2: Pro Features (Optional)
- **Cloud Sync**: Direct integration with OneDrive/Google Drive.
- **Advanced Export**: Export to PDF or high-fidelity HTML templates.
- **Formula Support**: A premium engine for basic Excel formula evaluation.

## 🏁 Final Step for Developer
1. Run `npm run compile`.
2. Run `vsce package`.
3. Upload the `.vsix` to the [VS Code Marketplace Management Console](https://marketplace.visualstudio.com/manage).
